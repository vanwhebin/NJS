## 指令

| 名称          | 示例                 | 说明                    |
| ------------ | -------------------- | ----------------------- |
| ping + url   | ping www.baidu.com   | ping 一个网址            |
| qrcode + url | qrcode www.baidu.com | 给网址生成一个二维码      |
| whois + url  | whois www.baidu.com  | 查询一个网址的 whois 信息 |
